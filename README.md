##  Completed Projects & Missions
Here are the foundational systems I have successfully built and tested so far:

### 1.  Aircraft Takeoff Decision System (`Week-01/Day-02/aircraft_takeoff_decision.py`)
* **What it does:** Evaluates whether an aircraft can safely take off based on runway length and aircraft weight.
* **Key Feature:** Includes a high-priority safety override that delays takeoff if visibility drops below 1000 meters, regardless of other conditions.

### 2.  Rocket Launch Decision System (`Week-01/Day-02/rocket_launch_decision.py`)
* **What it does:** Analyzes environmental factors (wind speed and rain) to approve, delay, or cancel a rocket launch.
* **Key Feature:** Features robust error-handling to catch text input errors and flags invalid negative wind speeds.

### 3.  Aircraft Speed Classifier (`Week-01/Day-02/aircraft_speed_classifier.py`)
* **What it does:** Takes real-time Mach speed inputs and instantly categorizes the flight regime into Subsonic, Transonic, Supersonic, or Hypersonic.

### 4.  Circle Geometry Calculator (`Week-01/Day-01/circle_calculator.py`)
* **What it does:** Accepts a radius input and uses the high-precision `math.pi` module to flawlessly compute a circle's diameter, circumference, and total area.
